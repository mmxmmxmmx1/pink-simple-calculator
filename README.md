# 🎀 粉紅色簡易計算機（Pink Simple Calculator）

這是一款以 Java 撰寫的桌面端簡易計算機，擁有可愛的粉紅色視覺介面，支援四則運算。

---

## ✨ 功能特色

- 支援加、減、乘、除基本運算
- 採用 Java Swing 開發 GUI
- 粉紅色主題介面，介面簡潔清晰
